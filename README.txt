FGCE Project Manager
====================

Giới thiệu
----------
FGCE Project Manager là ứng dụng quản lý dự án nội bộ dành cho tổ chức FGCE, hỗ trợ theo dõi tiến độ, xuất báo cáo PDF, Excel và biểu đồ trực quan.

Yêu cầu hệ thống
---------------
- Windows 10 hoặc 11 (64-bit)
- Không cần cài đặt Python hoặc phần mềm bổ sung
- Bộ cài đã tích hợp đầy đủ thư viện và wkhtmltopdf

Cài đặt
-------
1. Chạy file `FGCEProjectManagerSetup.exe`
2. Chọn thư mục cài đặt (mặc định: C:\Program Files\FGCEProjectManager)
3. Sau khi cài xong, ứng dụng sẽ tự khởi chạy

Sử dụng
-------
- Ứng dụng sẽ mở trình duyệt tại địa chỉ: http://localhost:5000
- Đăng nhập bằng tài khoản admin (tạo sẵn hoặc khởi tạo qua giao diện)
- Quản lý dự án, tiến độ, và xuất báo cáo PDF/Excel
- File dữ liệu `projects.db` sẽ được lưu tại:
  C:\Users\<Tên người dùng>\FGCEProjectManagerData

Tính năng chính
---------------
- Quản lý danh sách dự án, tiến độ, trạng thái
- Xuất báo cáo PDF theo mẫu đẹp, có logo và ngày xuất
- Xuất Excel tổng hợp tiến độ và trạng thái
- Biểu đồ tiến độ bằng matplotlib
- Giao diện đơn giản, dễ sử dụng

Gỡ cài đặt
----------
- Vào Control Panel → Programs → Uninstall FGCE Project Manager
- Hoặc dùng shortcut "Uninstall FGCE Project Manager" trong Start Menu

Hỗ trợ
------
Nếu gặp lỗi hoặc cần hỗ trợ kỹ thuật, vui lòng liên hệ bộ phận IT nội bộ FGCE.
